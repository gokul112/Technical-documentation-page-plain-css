React Introduction

What is React? React is a JavaScript library for building user interfaces.
  • React seperates the functions as components
  • React describes the structure of Web pages using modules
  • React elements are the building blocks of React pages
  • React elements are represented by tags
  • React tags label pieces of content such as "heading", "paragraph", "table", and so on
  • Browsers do not display the React tags, but use them to render the content of the page

A Simple React Document


*** EXAMPLE CODE ***
<!DOCTYPE React> <React> <head> <title> Page Title Goes Here </title> </head> <body> <h1> My First Heading </h1> <p> My first paragraph. <p> </body> </React>

Example Explained

  • The <!DOCTYPE React> declaration defines this document to be React5
  • The <React> element is the root element of an React page
  • The <head> element contains meta information about the document
  • The <title> element specifies a title for the document
  • The <body> element contains the visible page content
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

React Tags

React tags are element names surrounded by angle brackets:

<tagname> content goes here.. </tagname>
  • <React> tags normally come in pairs like <p> and </p>
  • The first tage in a pair is the start tag, the second tag is the end tag
  • The end tag is written like the start tag, but with a forward slash inserted before the tag name
TIP : The start tag is also called the opening tag, and the end tag the closing tag.
